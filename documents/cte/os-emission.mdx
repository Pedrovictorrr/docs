# CT-e OS - Conhecimento de Transporte Eletrônico - Outros Serviços

## O que é CT-e OS?

O **CT-e OS** (Conhecimento de Transporte Eletrônico - Outros Serviços) é um documento fiscal eletrônico utilizado para documentar prestações de serviços de transporte que não se enquadram nos demais tipos de CT-e.

## Características do CT-e OS

- **Finalidade**: Documentar serviços de transporte diversos
- **Validade**: Documento com validade jurídica
- **Obrigatoriedade**: Obrigatório para prestadores de serviços de transporte
- **SEFAZ**: Transmitido eletronicamente para a SEFAZ

## Tipos de Serviços Cobertos

### **Transporte de Cargas**
- Cargas gerais
- Cargas perigosas
- Cargas refrigeradas
- Cargas especiais

### **Serviços de Transporte**
- Frete rodoviário
- Frete aéreo
- Frete marítimo
- Frete ferroviário

### **Outros Serviços**
- Armazenagem
- Logística
- Distribuição
- Serviços auxiliares

## Estrutura do CT-e OS

### **Dados do Emitente**
- CNPJ/CPF do prestador de serviços
- Inscrição estadual
- Dados de endereço
- Dados de contato

### **Dados do Tomador**
- CNPJ/CPF do contratante
- Inscrição estadual
- Endereço de entrega
- Dados de contato

### **Dados do Serviço**
- Descrição do serviço
- Valor do frete
- Impostos incidentes
- Condições de pagamento

### **Dados de Transporte**
- Tipo de veículo
- Placa do veículo
- Dados do motorista
- Rota de transporte

```bash

  -H "Authorization: Bearer SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{

```

## Status do Documento

### **Fluxo de Status**

1. **PENDENTE**: Documento recebido e aguardando processamento
2. **PROCESSANDO**: Em processamento na SEFAZ
3. **AUTORIZADO**: Documento autorizado pela SEFAZ
4. **REJEITADO**: Documento rejeitado pela SEFAZ
5. **CANCELADO**: Documento cancelado (se aplicável)

### **Consulta de Status**

```bash
curl -X GET "https://api.managersaas.com/v1/cte/os/123456/status" \
  -H "Authorization: Bearer SEU_TOKEN"
```

## Cancelamento

### **Condições para Cancelamento**

- Documento deve estar autorizado
- Prazo máximo de 24 horas após autorização
- Justificativa obrigatória

### **Solicitação de Cancelamento**

```bash
curl -X POST "https://api.managersaas.com/v1/cte/os/123456/cancelar" \
  -H "Authorization: Bearer SEU_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "justificativa": "Erro na digitação dos dados"
  }'
```

## Arquivos Disponíveis

### **Download de Arquivos**

- **XML**: Arquivo XML original da SEFAZ
- **PDF**: Documento auxiliar em formato PDF
- **Log**: Log completo do processamento

```bash
# Download do XML
curl -X GET "https://api.managersaas.com/v1/cte/os/123456/xml" \
  -H "Authorization: Bearer SEU_TOKEN" \
  -o cte_os_123456.xml

# Download do PDF
curl -X GET "https://api.managersaas.com/v1/cte/os/123456/pdf" \
  -H "Authorization: Bearer SEU_TOKEN" \
  -o cte_os_123456.pdf
```

## Validações Importantes

### **Dados Obrigatórios**
- CNPJ/CPF do emissor e tomador
- Inscrição estadual
- Endereços completos
- Descrição do serviço
- Valor do frete

### **Validações de Negócio**
- CNPJ/CPF válidos
- Inscrição estadual ativa
- Endereços válidos
- Valores positivos
- Datas válidas
