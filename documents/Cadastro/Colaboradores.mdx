# Cadastro de Colaboradores

O cadastro de colaboradores é responsável pelo gerenciamento completo dos funcionários da empresa.

## Visão Geral

O módulo de colaboradores permite:

- Controle de dados pessoais e profissionais
- Gestão de cargos e funções
- Controle de acesso ao sistema
- Histórico de atividades

## Como Acessar

1. Acesse o menu principal
2. Clique em **Cadastros**
3. Selecione **Colaboradores**

## Funcionalidades

### Cadastrar Novo Colaborador

Para cadastrar um novo colaborador:

1. Clique no botão **"Novo Colaborador"**
2. Preencha as informações pessoais:
   - **Nome Completo**
   - **CPF**
   - **RG**
   - **Data de Nascimento**
   - **Endereço**
   - **Telefone**
   - **E-mail**
3. Configure as informações profissionais:
   - **Matrícula**
   - **Cargo**
   - **Setor**
   - **Equipe**
   - **Data de Admissão**
   - **Salário**
4. Defina as permissões de acesso
5. Clique em **"Salvar"**

### Editar Colaborador

Para editar um colaborador:

1. Localize o colaborador na lista
2. Clique no ícone de **edição**
3. Modifique as informações necessárias
4. Clique em **"Salvar"**

### Inativar Colaborador

Para inativar um colaborador:

1. Acesse o cadastro do colaborador
2. Altere o status para **"Inativo"**
3. Informe a data de desligamento
4. Salve as alterações

<Note>
ℹ️ **Importante**: Colaboradores inativos mantêm o histórico de atividades preservado.
</Note>

## Campos do Formulário

### Dados Pessoais

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| Nome Completo | Texto | Sim | Nome completo do colaborador |
| CPF | Texto | Sim | Número do CPF |
| RG | Texto | Não | Número do RG |
| Data de Nascimento | Data | Sim | Data de nascimento |
| Telefone | Texto | Sim | Telefone de contato |
| E-mail | E-mail | Sim | E-mail corporativo |

### Dados Profissionais

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| Matrícula | Texto | Sim | Número único da matrícula |
| Cargo | Seleção | Sim | Cargo do colaborador |
| Setor | Seleção | Sim | Setor de lotação |
| Equipe | Seleção | Não | Equipe de trabalho |
| Data de Admissão | Data | Sim | Data de admissão |
| Salário | Moeda | Não | Salário atual |

## Relatórios

### Relatório de Colaboradores

Gera relatórios com:

- Lista completa de colaboradores
- Dados por setor/equipe
- Aniversariantes do mês
- Admissões e desligamentos

### Exportação

Os dados podem ser exportados em:
- PDF
- Excel
- CSV

## Permissões de Acesso

O sistema permite configurar diferentes níveis de acesso:

- **Administrador**: Acesso total ao sistema
- **Gerente**: Acesso ao seu setor e subordinados
- **Operador**: Acesso limitado às suas funções
- **Visualizador**: Apenas consulta

## Dicas e Boas Práticas

- Mantenha os dados sempre atualizados
- Utilize matrículas sequenciais ou padronizadas
- Defina permissões de acordo com as funções
- Faça backup regular dos dados
